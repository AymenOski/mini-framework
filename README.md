# Mini Framework

A lightweight JavaScript framework inspired by React's core concepts.

## Features

- **UseState Hook** - Manage component state
- **UseEffect Hook** - Handle side effects
- **Jsx Factory** - Create virtual DOM nodes
- **Custom Event System** - On/Emit pattern
- **Virtual DOM** - Efficient DOM rendering

## Getting Started

Open `public/index.html` in your browser to see the TodoMVC example.
